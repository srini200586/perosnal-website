# Personal Website

This is my personal portfolio website (static HTML/CSS/JS).

## Tech

- HTML (`index.html`)
- CSS (`css/styles.css`)
- JavaScript (`js/main.js`)
- Static assets in `assets/`

## Run locally

Open `index.html` in your browser, or serve the folder with any static server.

Example (Python):

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy

This repo includes `vercel.json` and can be deployed on Vercel as a static site.
